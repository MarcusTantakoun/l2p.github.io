Getting Started¶

Installing¶

L2P can be installed with pip:

pip install l2p

Using L2P¶

First things first, import the whole L2P library, or necessary modules (see L2P):

from l2p import *

L2P requires access to an LLM. Set up your LLM class and methods using the abstract LLM(ABC) class. In this case, we have set up OpenAI’s GPT models to our library for quickstart.

export OPENAI_API_KEY='YOUR-KEY' # e.g. OPENAI_API_KEY='sk-123456'
engine = "gpt-4o-mini"
api_key = os.environ.get('OPENAI_API_KEY')
openai_llm = OPENAI(model=engine, api_key=api_key)

Create your customised prompts using the PromptBuilder class. These prompt templates can be used for any of the extraction methods. This is an example of creating a prompt for type extraction:

prompt_builder = PromptBuilder()

role_desc = "Your role is to..."
tech_desc = "You must follow this strategy..."
ex_desc = "Here is an example..."
task_desc = "Here are the given information..."

type_extraction_prompt = PromptBuilder(
    role = role_desc,
    technique = tech_desc,
    examples = [ex_desc]
    task = task_desc
)

Build PDDL domain components using the DomainBuilder class. This is an example of extracting PDDL types:

domain = "The Blocksworld domain is..."

domain_builder = DomainBuilder()

domain_builder.formalize_types(
    model = openai_llm,
    domain_desc = domain,
    prompt_template = type_extraction_prompt
)

Build PDDL problem components using the TaskBuilder class. This is an example of extracting PDDL initial states:

problem = "There are 5 blocks..."

task_builder = TaskBuilder()

task_builder.formalize_initial_states(
    model = openai_llm,
    problem_desc = problem,
    prompt_template = task_prompt
)

Build LLM feedback components using the FeedbackBuilder class. This is an example of getting LLM feedback from types:

feedback_builder = FeedbackBuilder()

feedback_builder.type_feedback(
    model = openai_llm,
    domain_desc = domain,
    response = "Retry this..."
    feedback_template = type_feedback_template,
    feedback_type = "llm"
)

Below are actual runnable usage examples. This is the general setup to build domain predicates:

 1import os, json
 2from openai import OpenAI
 3from l2p.domain_builder import DomainBuilder
 4from l2p.llm_builder import GPT_Chat
 5from l2p.utils.pddl_parser import prune_predicates, format_types
 6
 7def load_file(file_path):
 8    _, ext = os.path.splitext(file_path)
 9    with open(file_path, 'r') as file:
10        if ext == '.json': return json.load(file)
11        else: return file.read().strip()
12
13domain_builder = DomainBuilder()
14
15client = OpenAI(api_key=os.environ.get('OPENAI_API_KEY', None)) # REPLACE WITH YOUR OWN OPENAI API KEY
16model = GPT_Chat(client=client, engine="gpt-4o-mini")
17
18# load in assumptions
19domain_desc = load_file(r'tests/usage/prompts/domain/blocksworld_domain.txt')
20formalize_predicates_prompt = load_file(r'tests/usage/prompts/domain/extract_predicates.txt')
21types = load_file(r'tests/usage/prompts/domain/types.json')
22action = load_file(r'tests/usage/prompts/domain/action.json')
23
24# extract predicates via LLM
25predicates, llm_output = domain_builder.formalize_predicates(
26    model=model,
27    domain_desc=domain_desc,
28    prompt_template=formalize_predicates_prompt,
29    types=types,
30    nl_actions={action['action_name']: action['action_desc']}
31    )
32
33# format key info into PDDL strings
34predicate_str = "\n".join([pred["clean"].replace(":", " ; ") for pred in predicates])
35
36print(f"PDDL domain predicates:\n{predicate_str}")

The following output is:

### OUTPUT
(holding ?a - arm ?b - block) ;  true if the arm ?a is holding the block ?b
(on_top ?b1 - block ?b2 - block) ;  true if the block ?b1 is on top of the block ?b2
(clear ?b - block) ;  true if the block ?b is clear (no block on top of it)
(on_table ?b - block) ;  true if the block ?b is on the table
(empty ?a - arm) ;  true if the arm ?a is empty (not holding any block)

Here is how you would setup a PDDL problem:

 1from l2p.task_builder import TaskBuilder
 2
 3task_builder = TaskBuilder()
 4
 5# load in assumptions
 6problem_desc= load_file(r'tests/usage/prompts/problem/blocksworld_problem.txt')
 7formalize_task_prompt = load_file(r'tests/usage/prompts/problem/extract_task.txt')
 8
 9# extract PDDL task specifications via LLM
10objects, initial_states, goal_states, llm_response = task_builder.formalize_task(
11    model=model,
12    problem_desc=problem_desc,
13    prompt_template=formalize_task_prompt,
14    types=types,
15    predicates=predicates
16    )
17
18# format key info into PDDL strings
19objects_str = task_builder.format_objects(objects)
20initial_str = task_builder.format_initial(initial_states)
21goal_str = task_builder.format_goal(goal_states)
22
23# generate task file
24pddl_problem = task_builder.generate_task("blocksworld_problem", objects_str, initial_str, goal_str)
25
26print(f"PDDL problem: {pddl_problem}")
